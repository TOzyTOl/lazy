# 小蓝的博客 — lan-blog.dev

> 一款基于浏览器的 ARG（侵入式现实游戏）网络安全解谜游戏。
> 
> 你是一名调查员。佛山大学学生小蓝在参加「白帽子夜谈」线上直播后离奇失踪。
> 他的母亲请求你进入他的个人博客寻找线索。
> 你打开了他的博客——一个 2005 年的老式个人网站。
> 
> 有些秘密藏在表面之下。有些频率只有你能听到。

---

## 🎮 如何游玩

1. 用浏览器打开 `index.html`
2. 像浏览真实网站一样操作——点击链接、查看源代码、修改 URL
3. 发现线索，解开密码，层层深入
4. 真相在数据中等你

**不需要任何工具。只有你和浏览器。**

---

## 🔧 技术

纯静态网页。HTML + CSS + JavaScript。无框架、无后端、零依赖。

| 文件 | 说明 |
|------|------|
| `index.html` | 入口 |
| `css/retro.css` | 2005 年复古样式 |
| `js/state.js` | 游戏状态 |
| `js/router.js` | 路由 + 谜题逻辑 |
| `js/terminal.js` | 终端系统 |
| `js/effects.js` | 音效 + 视觉 + 结局 |
| `content/pages.js` | 页面内容 |

---

## 📡 部署

直接打开 `index.html` 即可本地游玩（摄像头功能需 HTTP 服务器）。

推荐部署到 GitHub Pages 以获得完整体验：

1. Fork 本仓库
2. Settings → Pages → Source: `main` → Save
3. 访问 `https://你的用户名.github.io/仓库名/`

---

## ⚠️ 注意

- 摄像头权限仅用于结局序列，不会上传任何数据
- 使用 `sessionStorage` 保存进度，关闭标签页即清除
- 建议佩戴耳机游玩
